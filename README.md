# sebastianremar.com

My personal website. Portfolio, resume, contact info, that kind of stuff.

## Tech Stack

- React + Vite
- Chakra UI v3
- react-i18next (English/Spanish)
- Deployed on Vercel

## Running Locally

```bash
npm install
npm run dev
```

Create a `.env` file with:

```
VITE_CAL_URL=your-cal-url
VITE_EMAIL=your-email
VITE_GITHUB_URL=your-github
VITE_LINKEDIN_URL=your-linkedin
```
